# 콕핏 사용설명서 (아주 쉬운 버전)

> 어려운 말 없이, 한 번에 이해되게 썼습니다.

---

## 0. 콕핏이 한마디로 뭔가요?

당신의 Hames(작업 공간 16개 + AI 비서 팀)를 **자동차 계기판처럼 한 화면에서 보고 조작**하는 도구입니다.

비유: 방이 16개인 집에서, 매번 각 방에 들어가 확인하는 대신 **현관 모니터 하나로 집 전체를 보는 것**과 같습니다.

보는 곳: 브라우저 주소창에 **`http://127.0.0.1:8765`** (지금 새 창에 떠 있습니다)

---

## 1. 켜기 / 끄기

| 하고 싶은 것 | 방법 |
|---|---|
| 켜기 | 저에게 **"콕핏 켜줘"** 라고 말씀 (또는 이미 켜져 있음) |
| 보기 | 브라우저에서 `http://127.0.0.1:8765` 열기 |
| 끄기 | 저에게 **"콕핏 꺼줘"** |

> 켜둬도 아무 문제 없습니다. 내 컴퓨터에서만 보이고, 남은 못 봅니다.

---

## 2. 화면은 4부분입니다

```
┌─────────────────────────────────────────────┐
│  ① 맨 위: 요약 숫자판 (한눈 상태)              │
├──────────┬───────────────────────┬──────────┤
│ ② 왼쪽    │   ③ 가운데              │ ④ 오른쪽  │
│  메뉴     │   (고른 메뉴 내용)       │ 실시간    │
│  버튼들   │                        │ 활동      │
└──────────┴───────────────────────┴──────────┘
```

- **① 맨 위(요약 숫자판):** 지금 어느 작업공간이 활성인지, 방은 몇 개인지, 오늘 한 일은 몇 번인지 등.
- **② 왼쪽(메뉴):** 보고 싶은 걸 고르는 버튼들.
- **③ 가운데:** 왼쪽에서 고른 메뉴의 내용.
- **④ 오른쪽(실시간 활동):** 지금 어디서 무슨 일이 벌어지는지 흐르듯 보여줌.

---

## 3. 왼쪽 메뉴 7개 — 각각 뭘 보나

| 메뉴 | 일상어로 |
|---|---|
| **Overview** | 전체 요약 — 한눈에 보기 |
| **Workspaces** | 작업공간(방) 목록 — 어디가 바쁘고 어디가 잠겨있나 |
| **Agents** | 내 AI 비서 팀 명단 (재무·전략·브랜드·기술·마케팅) |
| **Skills** | 쓸 수 있는 명령(기능) 목록 |
| **Harness** | 안전장치 기록 — 시스템이 막거나 허용한 일들 |
| **Git** | 코드 저장 상태 |
| **Operations** | ⭐ **AI 비서를 불러 일 시키는 곳** (아래 4번) |

---

## 4. ⭐ 제일 중요 — AI 비서 불러 쓰기 (Operations)

이게 이번에 새로 생긴 핵심 기능입니다. 5단계면 됩니다:

1. 왼쪽에서 **`Operations`** 클릭
2. **비서 고르기** (예: `CFO`=재무 담당, `CTO`=기술 담당, `CSO`=전략, `Marketer`=마케팅)
3. **시킬 일 적기** (예: "이번 달 유튜브 비용을 분석해줘")
4. **Launch** 누르기 → "정말 실행할까요?" → **예**
5. 아래 화면에 비서가 **일하는 과정이 실시간으로** 뜨고, 끝나면 **분석 결과**가 나옵니다.

> ⚠️ 지금은 비서가 **"읽고 분석해서 보고만"** 합니다. 당신 파일을 직접 고치거나 명령을 실행하진 **않습니다** (안전 모드). 진행 중인 비서는 **Stop** 버튼으로 멈출 수 있습니다.

---

## 5. 버튼 뜻 (헷갈리는 것만)

| 버튼 | 뜻 |
|---|---|
| **LIVE** | 켜면 화면이 5초마다 저절로 새로고침 (실시간 감시) |
| **⟳ REFRESH** | 지금 한 번 새로고침 |
| **⇱ open** | 그 작업공간 폴더를 Finder로 열기 |
| **⟱ git fetch** | 인터넷 최신 상태 받아보기 (안전, 아무것도 안 바꿈) |
| **Lock / Unlock** | 작업공간 잠그기 / 풀기 |
| **save / pull** (회색) | 아직 잠긴 기능 — 나중에 켤 수 있음 |

---

## 6. 색깔 뜻

- 🟢 **초록** = 정상 / 완료
- 🔴 **빨강** = 차단 / 오류
- ⚪ **회색 또는 "Phase 2" 표시** = 아직 안 켠 기능

---

## 7. 자주 묻는 것

| 질문 | 답 |
|---|---|
| 화면이 안 떠요 / 멈춰요 | **REFRESH** 누르기. 그래도 안 되면 "콕핏 다시 켜줘" |
| AI 비서가 내 파일 망가뜨리나요? | **아니요.** 지금은 읽고 분석만 합니다. |
| 남이 내 화면을 볼 수 있나요? | **아니요.** 내 컴퓨터에서만 보입니다 (`127.0.0.1`). |
| "오늘 한 일"은 뭘 세나요? | 당신이 어느 작업공간에서든 파일을 저장/수정한 횟수입니다. |
| 비서한테 뭘 시킬 수 있나요? | 분석·검토·조사 같은 **생각하는 일**. 예: "이 전략의 리스크 짚어줘" |
